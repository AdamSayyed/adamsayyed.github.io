Missile Trajectory Simulation & Optimization

C++, OpenGL, Numerical Methods

1. Project Overview

Developed a high-fidelity missile flight simulation using C++ to model complex aerodynamic forces, thrust profiles, gravity, and wind effects. The core physics engine utilizes Runge-Kutta 4 (RK4) numerical integration for precise trajectory calculation.

The project features a real-time 3D visualization engine built with OpenGL and GLFW, allowing users to view the flight path from multiple angles with interactive camera controls. Additionally, an angle-sweep optimization routine was implemented to automatically determine the optimal launch angle for maximum range.

2. Key Features

  • Physics Engine: Implemented RK4 integration to solve differential equations of motion, accounting for drag, thrust, and environmental factors.
  • 3D Visualization: Real-time rendering of the missile trajectory and environment using OpenGL. Includes animated flight playback.
  • Interactive Controls: User-controlled camera with pitch, yaw, and zoom capabilities (GLFW) to inspect the simulation from any perspective.
  • Trajectory Optimization: Automated routine that sweeps through launch angles to compute and visualize the maximum-range trajectory.

3. Technical Skills & Technologies

  • C++: Core physics modeling, simulation logic, and performance optimization.
  • OpenGL & GLFW: Real-time 3D rendering pipeline, window management, and input handling.
  • Numerical Methods: Runge-Kutta (RK4) integration for solving ordinary differential equations (ODEs).
  • Linear Algebra & Kinematics: Vector mathematics for force application, velocity updates, and coordinate transformations.
  • Simulation Architecture: Modular design separating physics calculation from rendering logic.
  • Git/GitHub: Version control and project management.
